# TextField

사용자로부터 한 줄의 텍스트 입력을 받기 위한 컴포넌트입니다.

## 사용 예시

<TextField.Root placeholder="Enter your name" />

```tsx
import { TextField } from '@radix-ui/themes';

function MyTextFieldExample() {
  return (
    <TextField.Root placeholder="Enter your name" />
  );
}

export default MyTextFieldExample;
```

## Do / Don't

### Do

*   명확한 플레이스홀더 텍스트를 제공하세요.
*   필요한 경우 `type` 속성을 사용하여 입력 형식을 지정하세요.

### Don't

*   비밀번호와 같은 민감한 정보에는 `type="password"`를 사용하지 않고 일반 텍스트로 표시하지 마세요.
*   너무 긴 텍스트를 입력받는 데 사용하지 마세요. (TextArea 사용 고려)

## Props

| Prop        | Type     | Default   | Description                |
| :---------- | :------- | :-------- | :------------------------- |
| `placeholder`| `string` | `undefined` | 입력 필드의 플레이스홀더 텍스트입니다. |
| `size`      | `1 | 2 | 3`| `2`       | 입력 필드의 크기를 조절합니다.  |
| `variant`   | `classic | surface | soft`| `surface` | 입력 필드의 스타일을 조절합니다. |
| `type`      | `string` | `text`    | 입력 필드의 타입을 지정합니다. |

## 가이드

TextField 컴포넌트는 사용자 입력을 위한 기본적인 요소입니다. 사용자에게 명확한 지침을 제공하고, 입력 유효성 검사를 통해 데이터 무결성을 확보하는 것이 중요합니다.
