# Button

사용자의 클릭을 통해 특정 액션을 실행하는 데 사용되는 컴포넌트입니다.

## 사용 예시

<Button>Click me</Button>

```tsx
import { Button } from '@radix-ui/themes';

function MyButtonExample() {
  return (
    <Button>Click me</Button>
  );
}

export default MyButtonExample;
```

## Do / Don't

### Do

*   명확하고 간결한 레이블을 사용하세요.
*   주요 액션에는 눈에 띄는 스타일을 적용하세요.

### Don't

*   너무 많은 버튼을 한 화면에 배치하지 마세요.
*   버튼의 용도를 모호하게 만들지 마세요.

## Props

| Prop      | Type     | Default   | Description                |
| :-------- | :------- | :-------- | :------------------------- |
| `children`| `ReactNode`| `undefined` | 버튼 내부에 렌더링될 내용입니다. |
| `size`    | `1 | 2 | 3`| `2`       | 버튼의 크기를 조절합니다.  |
| `variant` | `solid | soft | outline | ghost`| `solid`   | 버튼의 스타일을 조절합니다. |

## 가이드

버튼은 사용자가 애플리케이션과 상호작용하는 핵심 요소입니다. 일관된 디자인과 명확한 레이블링을 통해 사용자 경험을 향상시킬 수 있습니다.
